Slideshow Creator Manual

Menu

The menu is located at the top of the screen. Each option is listed under its relevant menu tag.

File

New

Creates a new slide show.

Save

Opens a dialog where the user can select a file to which the slideshow will be saved.

Load

Opens a dialog where the user can select a file from which to load a slide show.

Exit

Exits the slideshow program.

Help

About

Opens a dialog box containing information about the author, date of last update, and versioniong of the program.

Manual

Opens this webpage containing helpful information about usage of the program.

Control Toolbar

The Control Toolbar is located at the bottom of the screen. The instructions provided name the selectors from left to right.

Add Image Selector

Opens a dialog where the user can select one or more images to add to the slide show.

Remove Image Selector

Removes the image currently in view from the slide show.

First Slide Selector

The first slide selector changes the current slide to the first slide in the slide show.

Previous Slide Selector

The previous slide selector changes the current slide to the previous slide in the slide show. If the current slide is the first slide, it will change the current slide to the last slide.

Next Slide Selector

The next slide selector changes the current slide to the next slide in the slide show. If the current slide is the last slide, it will change the current slide to the first slide.

Last Slide Selector

The last slide selector changes the current slide to the last slide in the slide show.

Fullscreen Toggle

Toggles fullscreen window size.

Thumbnail Overview Toggle

Toggles an overview of all slides in the slide show where all slides are displayed as thumbnails.
